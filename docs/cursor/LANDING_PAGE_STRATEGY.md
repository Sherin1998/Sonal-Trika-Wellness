# LANDING PAGE STRATEGY — Trika Yoga & Wellness

Dedicated landing pages drive high-value conversions. Do not funnel all traffic through a single Services page.

Each landing page follows the **problem-first content structure**:

1. Problem → 2. Symptom → 3. Desired transformation → 4. Methodology → 5. CTA

---

## Executive Wellness

**URL:** `/executive-nervous-system-reset`

**Target Audience:** Founders · CXOs · Investors

**Primary CTA:** Schedule Executive Consultation

**Emotional Goal:** Recognition that leadership performance requires nervous system recovery, not more hustle.

**Business Goal:** High-ticket executive wellness engagements.

### Content Angle

Open with chronic hyper-arousal in leadership — decision fatigue, sleep fragmentation, emotional reactivity under pressure. Position the executive reset as physiological recovery, not another wellness perk.

### Section Structure

1. Dark cinematic hero — minimal copy, huge typography
2. Problem block — "The nervous system of a leader under sustained pressure"
3. Symptom grid — sleep, focus, reactivity, recovery (asymmetric layout)
4. Transformation — what restored regulation feels like in boardroom and life
5. Methodology — gong immersions, breathwork, private sessions tailored for executives
6. Social proof — executive testimonial (polaroid style)
7. CTA — Schedule Executive Consultation

## Landing Page Design Rules

1. **Unique layout per landing page** — no shared template shell
2. **Problem-first copy** on every section above the fold
3. **One primary CTA** repeated max 2 times (mid-page + footer)
4. **No navigation clutter** — consider minimal header on conversion pages
5. **Awwwards test** before shipping — must not look like a generic landing page builder output
6. **Mobile-first** but desktop should feel editorial and spacious
